## Tentang Aplikasi Ini
JEA Image Resizer Pro adalah solusi cepat untuk kebutuhan manipulasi gambar massal. Dirancang menggunakan Python murni, aplikasi ini sangat ringan dan tidak membutuhkan proses instalasi yang rumit. 

Fitur Utama:
* Bulk Resize dalam hitungan detik.
* Mendukung format .jpg, .png, dan .webp.
* Antarmuka Dark Mode elegan.
